# Bachata Training

一個可直接用手機觀看的 Bachata 教學平台。

## 內容

- 8 週學習路線
- 招數心智圖與優先學習順序
- 16 個招數使用情境表
- 18 個 Bachata 教學影片與搜尋入口
- 收藏與練習進度紀錄

## 使用方式

開啟 `index.html` 即可使用。部署到 GitHub Pages 後，手機可直接用網址觀看。

## GitHub Pages

建議把 GitHub Pages 設定為：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`
